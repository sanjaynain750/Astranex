# AstraNex - AI-Powered Penetration Testing Suite

AstraNex is an all-in-one penetration testing toolkit with 9 powerful modules. It works on Linux, Termux, and Windows.

## ⚡ Features
- **9 Built-in Modules**: Port Scanner, Subdomain Finder, Web Fuzzer, Vuln Scanner, Secret Scanner, Container Scanner, WAF Detector, Auto Exploiter, Cloud Misconfig Scanner.
- **AI Integration**: Use local Ollama or OpenAI API to guide your testing.
- **Easy to Use**: Single command interface.
- **Fast**: Multi-threaded for maximum speed.
